How to download the required files

1
Download the PR diff

Open the Pull Request on GitHub. Append .diff to the URL and save the file.

https://github.com/owner/repo/pull/123.diff

Or use .patch for the email-style patch format.

2
Download the full source files optional

For each file changed in the PR, download the current version from GitHub:

  • Navigate to the file in the repo on GitHub.
  • Click the Raw button.
  • Save the page (Ctrl+S / Cmd+S).

Alternatively, clone the repo locally and upload the files directly from your machine. The more source files you upload, the more context the selected AI model has.

git clone https://github.com/owner/repo.git
cd repo && git checkout <pr-branch>
3
Upload here

Upload the .diff file in slot 1, add optional source or review documents in the remaining slots, then click RUN ANALYSIS.
